## ‚ö° Quick Start (15 Minutes)

### Step 1: Create Your Repository (2 minutes)

1. Go to [github.com/new](https://github.com/new)
2. **Repository name:** `SammyTourani` (must match your username EXACTLY)
3. Make it **Public**
4. ‚úÖ Check "Add a README file"
5. Click **Create repository**

### Step 2: Upload Your Files (5 minutes)

**Option A: Using GitHub Web Interface**
1. Go to your new repository
2. Click "uploading an existing file"
3. Drag and drop `README.md`
4. Commit with message: "Add epic profile README"

**Option B: Using Git (Recommended)**
```bash
# Clone the repository
git clone https://github.com/SammyTourani/SammyTourani.git
cd SammyTourani

# Copy your files
cp /path/to/README.md .
cp -r /path/to/.github .

# Commit and push
git add .
git commit -m "Add epic profile README with animations"
git push origin main
```

### Step 3: Enable GitHub Actions (3 minutes)

1. Go to your repository **Settings**
2. Click **Actions** ‚Üí **General** (left sidebar)
3. Under "Workflow permissions":
   - Select **"Read and write permissions"**
   - Check **"Allow GitHub Actions to create and approve pull requests"**
4. Click **Save**

### Step 4: Run the Snake Animation (2 minutes)

1. Go to **Actions** tab in your repository
2. Click **"Generate Snake Animation"** workflow
3. Click **"Run workflow"** ‚Üí **"Run workflow"**
4. Wait ~30 seconds for completion
5. Refresh your profile page

### Step 5: View Your Profile (3 minutes)

1. Go to `github.com/SammyTourani` (your profile)
2. Your README should be displayed!
3. Open in incognito mode to see as others do
4. Test on mobile device

## üîß Troubleshooting

### Issue: Repository not showing on profile

**Fix:**
- Repository must be named EXACTLY as your username: `SammyTourani`
- Repository must be Public
- README.md must be in root directory (not in a folder)

### Issue: Snake animation not appearing

**Fix:**
1. Go to Actions tab ‚Üí Check if workflow ran successfully
2. Settings ‚Üí Actions ‚Üí Verify "Read and write permissions"
3. Run workflow manually
4. Wait for completion (check for green checkmark)
5. Hard-refresh profile page (Ctrl+Shift+R / Cmd+Shift+R)

### Issue: Stats cards showing wrong username

**Fix:**
- Search for "YOUR_USERNAME" in README.md
- Replace ALL instances with: `SammyTourani`
- Commit and push changes

### Issue: Links not working

**Fix:**
- Check each link individually
- Verify LinkedIn URL format: `linkedin.com/in/sammy-tourani`
- Update any broken repo links
- Test all mailto: links
